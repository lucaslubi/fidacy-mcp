# Fidacy — Payment Firewall (OpenClaw plugin)

A signed, independently-verifiable verdict on every money-moving action your agent
takes. Blocks wrong/lookalike payee, over-cap, and duplicate-invoice fraud **before**
money moves. Non-custodial, local-first. Ships the `@fidacy/mcp` server with 5 tools:
`request_payment`, `verify_mandate`, `get_audit_proof`, `assess_action`, `upgrade`.

Verify any verdict yourself: https://api.fidacy.com/.well-known/jwks.json
